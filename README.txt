1. Execute the makefile using make utility

2. Change directory to 'run'

3. Change the parameters as per the requirement in 'input.dat'

4. Run the binary using the following command in the terminal:
    ../bin/lidDrivenCavity

5. Solver types:
    1 - BGK Raw Moments
    2 - MRT Central Moments
    3 - BGK Central Moments
