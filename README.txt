1. Before using the make utility in the project folder, run the following command:
    make clean

2. Execute the makefile using make utility

3. Change directory to 'run'

4. Change the parameters as per the requirement in 'input.dat'

5. Run the binary using the following command in the terminal:
    ../bin/lidDrivenCavity

6. Solver types:
    1 - BGK Raw Moments
    2 - MRT Central Moments
    3 - BGK Central Moments
